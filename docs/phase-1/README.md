# Filora — Phase 1 Documentation

Documentation for the Filora native Android file manager (appblish), covering the
Phase-1 milestones **M0–M8**. Phase 1 began as planning artifacts; the app has
since been built out, so this index also links the **as-built** docs.

New here? Start with the [root README](../../README.md) (build & run),
[architecture.md](architecture.md) (how the code is structured), and the
[CHANGELOG](../../CHANGELOG.md) (what shipped).
